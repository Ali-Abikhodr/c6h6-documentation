# www.c6h6.org

www.c6h6.org allows not only to store all the information related to a sample but also to process the experimental results.

Even it is looks quite chemically oriented the database is able to store any kind of information about a sample. A sample could be a biological tissue, a polymer, a chemical compound, etc.

Among the processing that can be achieved online we may cite:

- NMR spectrum assignment
- IR spectrum assignment
- Mass spectrum assignment
- Image analysis (like SEM / TEM)
- ...
